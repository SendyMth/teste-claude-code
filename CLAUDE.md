## Project Overview

A Python project using Streamlit for the UI, yfinance for financial data, Plotly for charting, and Pandas for data manipulation.

## Setup & Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Tech Stack

- **UI**: Streamlit
- **Data**: yfinance, Pandas
- **Visualization**: Plotly
